# **App Name**: CyberPrompt Duel

## Core Features:

- Player Authentication: Secure player login/registration with real-time status.
- Challenge Generation: Algorithmically generate diverse prompt injection challenges based on difficulty level.
- Real-time Gameplay: Enable players to compete head-to-head in real-time through prompt submissions.
- AI Judge: The AI will evaluate each submission, with its evaluation acting as a tool the app will incorporate, determining a winner based on correctness, creativity, and prompt manipulation.
- Dynamic Scoring: Implement a points system that rewards innovative or unexpected solutions.
- Leaderboard: Track and display player scores and rankings in real-time.
- Level Progression: Allow players to advance through increasingly difficult levels as they improve their skills.

## Style Guidelines:

- Primary color: Cyan (#00FFFF) to align with the futuristic cyber aesthetic.
- Background color: Dark slate gray (#2F4F4F), desaturated and dark, to provide contrast and set a futuristic mood.
- Accent color: Lime green (#32CD32), analogous to cyan, for interactive elements and highlights.
- Heading font: 'Orbitron' sans-serif; Body font: 'Poppins' sans-serif. Note: currently only Google Fonts are supported.
- Use neon-glowing icons that represent code, AI, and hacking concepts.
- Fullscreen layout with split-sections for gameplay and interactive levels.
- Implement neon glow effects, hover light pulses, and glitch transitions.
